## Activity 5
**Functional Requirements:** The search feature accepts a search term from the user and returns courses meeting filter in a table.

**Non-functional Requirements:** The search feature returns results in under 1 minute.

I would look to improve the functional requirment by allowing a user to see pre-requisite data directly in table.
I would look to improve the non-functional requirment by returning results in under 30 seconds.
